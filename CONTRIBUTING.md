# Contributing to Azure Beginners Lab

First of all — thank you for wanting to help! 🙏

This project is built for beginners, so contributions of all kinds are welcome:
fixing typos, improving explanations, adding screenshots, or suggesting new labs.

---

## Ways to contribute

### 🐛 Report a problem
If a step doesn't work or something is confusing:
1. Click **Issues** at the top of this repo
2. Click **New Issue**
3. Describe what happened and which lab you were on

### ✏️ Fix a typo or improve wording
1. Click the file you want to edit
2. Click the pencil icon ✏️ in the top-right
3. Make your change
4. Click **Propose changes**
5. Submit a Pull Request

### 💡 Suggest a new lab
Open an Issue with the title: `[Lab Idea] Your idea here`
Describe what the lab would teach and why it's useful for beginners.

---

## Guidelines

- **Keep it beginner-friendly** — explain the why, not just the what
- **One concept per lab** — don't try to teach everything at once
- **Always include a cleanup step** — never leave someone with surprise charges
- **Use plain English** — avoid jargon unless you explain it first

---

## Code of conduct

Be kind. This is a learning space.
Everyone here was a beginner once — including me.
